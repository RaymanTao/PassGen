# 密码生成器 (Password Generator)

一个安全、易用的在线密码生成工具，帮助您快速生成强密码，提升账户安全性。

## ✨ 功能特性

### 核心功能
- 🔐 **安全密码生成** - 使用 Web Crypto API 生成加密安全的随机密码
- 📏 **自定义长度** - 支持 4-128 位密码长度
- 🎛️ **灵活配置** - 可选择包含/排除大写字母、小写字母、数字、特殊字符
- 💪 **强度评估** - 实时显示密码强度等级和评分（0-100分）
- 📋 **一键复制** - 快速复制密码到剪贴板
- 👁️ **可见性切换** - 显示/隐藏密码内容

### 高级功能
- 🎯 **预设选项** - 快速选择常用密码类型（强密码、中等密码、PIN码等）
- 🚫 **排除选项** - 排除相似字符（0/O, 1/l/I）和歧义字符
- 📊 **统计信息** - 显示密码长度、各字符类型数量统计
- 🌓 **主题切换** - 支持深色/浅色主题，自动保存偏好设置
- 📱 **响应式设计** - 完美适配桌面端和移动端

### 安全特性
- ✅ 所有密码在客户端本地生成，不上传服务器
- ✅ 不记录、不存储用户生成的密码
- ✅ 使用加密安全的随机数生成器
- ✅ 遵循安全最佳实践

## 🚀 快速开始

### 本地运行

1. 克隆或下载项目
2. 直接在浏览器中打开 `index.html` 文件

或者使用本地服务器：

```bash
# 使用 Python
python -m http.server 8000

# 使用 Node.js (需要安装 http-server)
npx http-server

# 使用 PHP
php -S localhost:8000
```

然后在浏览器中访问 `http://localhost:8000`

## 📖 使用说明

### 基础使用

1. **选择密码长度** - 使用滑块调整密码长度（4-128位）
2. **选择字符类型** - 勾选需要包含的字符类型
3. **生成密码** - 点击"生成密码"按钮或使用预设选项
4. **复制密码** - 点击复制按钮将密码复制到剪贴板

### 预设选项

- **强密码 (16位)** - 包含所有字符类型，适合重要账户
- **中等密码 (12位)** - 包含字母和数字，适合一般账户
- **PIN码 (6位)** - 纯数字，适合临时验证码
- **仅字母** - 仅包含大小写字母
- **仅数字** - 仅包含数字

### 高级选项

- **排除相似字符** - 排除容易混淆的字符（如 0/O, 1/l/I）
- **排除歧义字符** - 排除可能引起问题的特殊字符

### 键盘快捷键

- `Ctrl/Cmd + Enter` - 生成新密码
- `Ctrl/Cmd + C` - 复制当前密码（当密码已生成时）
